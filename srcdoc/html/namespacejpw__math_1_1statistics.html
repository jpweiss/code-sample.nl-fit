unsigned  s2,
bool  verbose = true 
)

Overloaded version that takes a seed as 2 unsigned integers.

The seed should be, at minimum, a 48-bit number.

Parameters:
s1The upper 16 bits of the seed, stored in the first 2 bytes of s1. In other words, treat this parameter as if it's an unsigned short.
s2The lower 32 bits of the seed.
verboseIf true, print out information about the seed to stdout.
